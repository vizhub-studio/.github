The core technology behind **VizHub**, a platform for building interactive data visualizations with **D3** & **React**.

## The Open Source Projects

- **vzcode**: An in-browser code editor built on [CodeMirror](https://codemirror.net/) and [ShareDB](https://share.github.io/sharedb/)
- **vizhub-runtime**: A secure sandbox for executing and rendering visualizations built on [Rollup](https://rollupjs.org/) and [Sucrase](https://www.npmjs.com/package/sucrase)
- **editcodewithai**: A one-shot LLM-driven source code editing tool

## Our Mission

Build the world's best open environment for learning, teaching, and creating data visualization.
